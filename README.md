# 🐾 Pet Shop Landing Page

Uma Landing Page moderna, rápida e responsiva desenvolvida para um Pet Shop. Este projeto tem como objetivo demonstrar os serviços, sobre a empresa, avaliações de clientes e contato de forma atrativa e direta, oferecendo uma excelente experiência para o usuário.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído com as melhores e mais modernas tecnologias do ecossistema front-end:

- **[Next.js](https://nextjs.org/)**: Framework React para renderização rápida, otimização de imagens e SEO.
- **[React](https://react.dev/)**: Biblioteca JavaScript para construção da interface de usuário.
- **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS utilitário para estilização rápida e responsiva.
- **[Shadcn UI](https://ui.shadcn.com/)**: Componentes de interface acessíveis e customizáveis.
- **[AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)**: Biblioteca para animações fluidas enquanto o usuário rola a página.
- **[Lucide React](https://lucide.dev/)**: Biblioteca de ícones bonitos e consistentes.
- **[Embla Carousel](https://www.embla-carousel.com/)**: Utilizado para criação de sliders fluidos.

## ✨ Funcionalidades

- **Design Responsivo**: Adaptado para funcionar perfeitamente em celulares, tablets e desktops.
- **Animações Fluidas**: Elementos surgem na tela suavemente ao rolar a página, melhorando o engajamento através da biblioteca AOS.
- **Otimização de Imagens**: Uso do componente `next/image` para carregamento rápido e economia de dados.
- **Botão Flutuante do WhatsApp**: Facilita o contato direto do cliente com o Pet Shop.
- **Seções Estruturadas**:
  - **Hero**: Área de destaque inicial com chamada para ação (Call to Action).
  - **Sobre (About)**: Informações sobre a empresa e infraestrutura.
  - **Serviços (Services)**: Demonstração do que o Pet Shop oferece (Banho, Tosa, Veterinário, etc).
  - **Depoimentos (Testimonials)**: Prova social com avaliações de clientes.
  - **Rodapé (Footer)**: Links úteis e informações de localização.

## 🛠️ Como rodar o projeto localmente

Siga os passos abaixo para rodar o projeto na sua máquina:

1. **Clone o repositório**
   ```bash
   git clone https://github.com/gabrieltomazi/petshop-nextjs.git
   ```

2. **Acesse a pasta do projeto**
   ```bash
   cd petshop
   ```

3. **Instale as dependências**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento**
   ```bash
   npm run dev
   ```

5. **Acesse no navegador**
   Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
